# 口腔内窥镜智能影像分析与三维重建系统 Mermaid 图

## 1. 系统总体架构图

```mermaid
flowchart TB
    A["口腔内窥镜设备"] --> B["移动端 APP<br/>Android / iOS"]
    A --> C["PC 端应用"]

    B --> D["图像采集与上传"]
    C --> D

    D --> E["预处理模块<br/>解码 / 归一化 / 增强 / 质量筛查"]
    E --> F["牙齿分割模块<br/>YOLO / CNN / OpenCV 后处理"]
    F --> G["2D 拼接模块<br/>特征提取 / 匹配 / 配准 / 融合"]
    G --> H["健康状态辅助分析模块<br/>龋坏 / 牙结石 / 牙龈异常"]
    H --> I["3D 重建模块<br/>多视图恢复 / 结构表达"]

    E --> J["模型推理服务"]
    F --> J
    G --> J
    H --> J
    I --> J

    J --> K["服务器与管理后台"]
    K --> K1["用户管理"]
    K --> K2["病例管理"]
    K --> K3["图像与结果存储"]
    K --> K4["模型版本管理"]
    K --> K5["日志与权限控制"]

    G --> L["2D 连续口腔图"]
    H --> M["辅助诊断结果"]
    I --> N["3D 牙弓模型"]

    L --> O["医生端查看"]
    M --> O
    N --> O

    L --> P["患者端查看"]
    M --> P
```
