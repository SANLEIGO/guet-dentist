# 口腔内窥镜拼接 v1 - 使用指南

## 功能特性

### 🔪 牙齿分割
- **深度学习方法**：使用 AlphaDent (YOLOv8) 进行牙齿区域检测
- **精细化处理**：可选 GrabCut 边界优化
- **可视化反馈**：绿色覆盖层直观显示分割结果
- **多视图展示**：并排对比、网格展示、单独查看三种模式
- **统计信息**：覆盖率、像素数、分割方法等详细指标

### 🔗 图像拼接
- **标准化流水线**：分割 → 特征提取 → 配准 → 融合
- **多特征方法**：支持 ORB、AKAZE、SIFT、LoFTR
- **质量评估**：清晰度、曝光度、匹配质量指标
- **诊断报告**：完整的 JSON 诊断数据

## 使用流程

### 1. 上传图像
- 在左侧侧边栏点击「上传多张口腔内窥镜图像」
- 支持 JPG、PNG、BMP、TIF 等格式
- 建议上传 2-4 张同一牙弓的图像

### 2. 调整参数
- **分割设置**：
  - 置信度阈值：0.01-0.5，默认 0.1
  - GrabCut 精细化：默认启用
- **拼接设置**：
  - 特征方法：ORB（推荐）、AKAZE、SIFT、LoFTR

### 3. 执行分割
- 点击「执行分割」按钮
- 查看分割结果对比（原图 vs 掩膜）
- 选择查看模式：
  - **并排对比**：逐张对比原图和分割掩膜
  - **网格展示**：所有图像网格化展示
  - **单独查看**：选择单张图像详细查看

### 4. 下载分割结果
- **所有掩膜 (ZIP)**：包含所有图像的掩膜和原始图
- **分割报告 JSON**：包含统计信息和方法详情

### 5. 开始拼接
- 确认分割效果满意后
- 点击「开始拼接」按钮
- 等待处理完成

### 6. 查看结果
- **拼接结果**：最终拼接的全景图
- **匹配可视化**：特征点匹配连线
- **质量指标**：清晰度、曝光度等数据
- **诊断信息**：完整的处理日志和参数

### 7. 下载结果
- **拼接图 PNG**：最终拼接结果
- **诊断 JSON**：完整的诊断数据

## 技术参数

### 分割
- **模型**：AlphaDent (YOLOv8)
- **输入尺寸**：960x960
- **默认置信度**：0.1
- **后处理**：GrabCut (3次迭代)

### 特征提取
- **ORB**：2000 特征点，scaleFactor=1.2
- **AKAZE**：默认参数
- **SIFT**：2500 特征点，对比度 0.02

### 配准
- **方法**：单应性矩阵 (RANSAC)
- **回退**：仿射变换
- **阈值**：4.0 像素
- **最少点数**：8 对

### 融合
- **方法**：高斯羽化
- **羽化半径**：30 像素

## 注意事项

1. **图像质量**：确保图像清晰、曝光适中
2. **重叠区域**：相邻图像需要有 30-50% 的重叠
3. **采集顺序**：按从左到右或从右到左的顺序采集
4. **当前限制**：仅支持 2 张图像的拼接
5. **环境变量**：需设置 `DENTAL_SEG_WEIGHTS` 指向模型权重路径

## 运行方法

```bash
# 设置环境变量
export DENTAL_SEG_WEIGHTS=/path/to/alphadent_9cls_960.pt

# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

## 故障排查

### 分割失败
- 检查环境变量 `DENTAL_SEG_WEIGHTS` 是否正确
- 确认模型文件存在
- 降低置信度阈值

### 拼接失败
- 确保两张图像有充分重叠
- 检查图像清晰度
- 尝试更换特征方法（如从 ORB 改为 SIFT）

### 性能优化
- 减少上传图像数量
- 降低图像分辨率
- 使用 ORB 特征方法（最快）
